Creación de la estructura de un proyecto React.

    npx create-react-app nombre-app
    cd nombre-app
    npm start

    Estructura de archivos.

    index.html = encargado de renderizar el contenido.
    Package.json = contiene info sobre el proyecto, librerías y dependencias.
    .gitignore = indica qué ficheros NO subir a GitHub.
    index.js = es el componente principal, se encarga de renderizar
    el contenido y los componentes.
    src = Directorio principal, aloja todos los archivos de los componentes.
